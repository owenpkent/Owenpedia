# Household Checklists

- [ ] Dishwasher unloaded
- [ ] Dishes washed

- [ ] Stove cleaned
- [ ] Boxes opened, put away, broken down
- [ ] Trash and recycling emptied
- [ ] Check mail
- [ ] Sweep floor
